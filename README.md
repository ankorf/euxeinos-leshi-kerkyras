# Εύξεινος Λέσχη Ποντίων Κέρκυρας

Επίσημος ιστότοπος της Ευξείνου Λέσχης Ποντίων Κέρκυρας — χορευτικός και πολιτιστικός σύλλογος από το 1997.

**Live site:** https://ankorf.github.io/euxeinos-leshi-kerkyras/

---

## Δομή Project

```
├── index.html              Single-page website
├── assets/
│   ├── css/style.css       Όλα τα styles
│   ├── js/main.js          Scroll effects, counters, mobile menu
│   └── images/
│       ├── logo.jpg        Λογότυπο / έμβλημα
│       ├── hero-bg.jpg     Φωτογραφία hero section
│       ├── qr-code.png     QR code πρόσβασης
│       └── gallery/        Φωτογραφίες gallery
├── docs/                   Έγγραφα πρότασης (PDF)
└── tools/                  Βοηθητικά scripts
```

## Τεχνολογίες

- HTML5 / CSS3 / JavaScript (vanilla)
- Google Fonts: Playfair Display + Inter
- Χωρίς framework, χωρίς build tools
- Hosting: GitHub Pages
